# Rolllink污水处理实验装置计算
## 一、设计流量
- 日平均流量
$Qd = 1m^3/d = 0.0417m^3/h = 41.7L/h = 0.0116L/s$
- Kz取1.40,最大日流量
$Qmax = Kz*Qd = 58.38L/h = 0.016L/s$


## 二、污水生物处理
### 2.1 进水条件
- 进入曝气池的平均流量$Q = 1m^3/d$,  
- 最大设计流量$0.016L/s$
- 污水中$BOD_5$浓度为250mg/L,实验装置无一级处理,则进入生化池中的污水$BOD_5$浓度为250mg/L
- 污水中TN浓度40mg/L, TP浓度5mg/L, 水温T = 20


### 2.2 设计参数
-   污水处理程度  
$n_{BOD} = (250-10)/250 = 0.96$

- $BOD_5$污泥负荷率  
$N_s = (K_2*S_n*f)/n = (0.02*10*0.75)/0.96 = 0.156 Kg BOD_5/(Kg MLSS*d)$  
式中：  
$K_2$——有机物最大比降解速率与饱和常数的比，一般采用0.0168～0.0281，取0.02；  
$S_n$——处理后出水中的$BOD_5$浓度,按一级A标准取10mg/L；  
$f$——MLVSS/MLSS值，一般采用0.7-0.8。  

- 曝气池內混合液污泥浓度
$X = (R*r*10^6)/((1+R)*SVI) = (0.5*1.2*10^6)/((1+0.5)*120)=3333 mg/L$  
式中：  
$R$——污泥回流比，取R=50%;  
$r$——系数;  
$SVI$——污泥容积指数，取SVI=120。  


### 2.3平面尺寸计算
#### 好氧段
- 曝气池有效容积，按污泥负荷计算  
$V=(24*Q*(S_o-S_e))/(1000*L_s*X)=1.53m^3$  
式中：  
$S_o$——进水BOD$(mg/L), 取250$;  
$S_e$——出水BOD$(mg/L)，取10$;  
$Q$——设计流量$(m^3/h)$;  
$L_s$——BOD污泥负荷$(kgBOD_5/kgMLSS·d)$;  
$X$——混合液悬浮固体平均浓度$(gMLSS/L)$;  
按曝气池个数N=2计算，
每组曝气池有效容积 $V_1 = V/N = 1.53/2 = 0.765 m^3$ 
- 单座曝气池面积
设计中有效水深取$H=0.3m$,
$F = V_1/H = 0.765/0.3 = 2.55 m^2$
- 曝气池长度
曝气池宽度取0.3m
$L = F/B = 2.55/0.3 = 8.5m$
曝气池设3条廊道，则每条廊道长$L_1 = L/3 = 2.83m$


#### 厌氧段
- 厌氧段容积  
$V_p=(t_p*Q)/24=0.0625m^3$  
式中：  
$t_p$——厌氧区水力停留时间$h$,宜为1～2，取1.5;  
$Q$——设计流量$(m^3/h)$;  
- 单座厌氧段体积  
$V_p = 0.0625/2 = 0.0313m^3$


#### 缺氧段
- 缺氧段容积  
$\Delta X_v=y*Y_t*(Q*(S_o-S_e)/1000)=0.75*0.8*(1*(250-10)/1000)=0.144m^3$  
$K_{de(T)}=K_{de(20)}*1.08^{T-20}=0.06$  
$V_n=(0.001*Q*(N_k-N_{te})-0.12*\Delta X_v)/(K_{de}*X)=0.038m^3$  
式中：  
$Q$——设计流量$(m^3/h)$;  
$N_k$——进水总凯氏氮$(mg/L)$，取40;  
$N_{te}$——出水总凯氏氮$(mg/L)$，取15;  
$\Delta X_v$——排出生物反应池的微生物量$(KgMLVSS/d)$;  
$K_{de}$——脱氮速率$(kgNO_3-N/kgMLSS·d)$，可采用0.03~0.06,取0.06;  
$y$——MLSS中MLVSS所占比例，取0.75;  
$Y_t$——总产率系数，有初沉池取0.3，无初沉池取0.6～1.0，该实验装置无初沉池取0.8;
